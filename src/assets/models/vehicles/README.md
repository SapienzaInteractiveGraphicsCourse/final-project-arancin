# Vehicle model assets

This directory contains imported vehicle assets only. Runtime vehicle logic should live outside this directory.

Current structure:

- `porsche-cayman-gt4/source`: GLB model.
- `porsche-cayman-gt4/textures`: texture maps used by the Porsche model.
- `nissan-silvia-kouki/source`: FBX model.
- `nissan-silvia-kouki/textures`: texture maps used by the Silvia model.

The procedural kart is intentionally not stored here. It should be built in JavaScript with Three.js primitives.

Before final delivery, document the source and license of each imported model and texture in the project report.
